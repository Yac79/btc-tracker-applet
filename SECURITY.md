## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **DO NOT** open a public issue or comment. Instead, report it privately to ensure the safety of all users.

You can report vulnerabilities by sending an email to:
👉 **dustroad@protonmail.com**

Please include the following details in your report:
- A description of the vulnerability and its potential impact.
- Steps to reproduce the issue (code snippets or screenshots are welcome).

Thank you for helping keep this project secure!
