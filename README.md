# Bitcoin Tracker Applet

A lightweight, privacy-focused applet for the Cinnamon desktop (Linux Mint) that allows you to monitor the Bitcoin price in real-time directly from your panel.

## Features
- Real-time price tracking via **CoinGecko API**.
- 24-hour price change percentage display.
- Color-coded indicators (green for up, red for down).
- Multi-currency support (configurable via settings).
- Built using native Cinnamon libraries (Soup), requiring no external dependencies like curl or jq.

## Privacy & Security
This applet is designed with a **privacy-first** approach:
- **No Tracking**: It does not collect or transmit any user data.
- **No API Keys**: Works out of the box without requiring personal accounts or private keys.
- **Native Execution**: Uses standard system libraries to ensure a minimal footprint and maximum security.

## Installation
1. Download the repository files.
2. Copy the folder `btc-tracker@ya3c` to:
   `~/.local/share/cinnamon/applets/`
3. Enable the applet from the "Applets" settings in Cinnamon.
